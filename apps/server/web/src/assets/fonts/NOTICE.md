# Bundled Fonts

The browser terminal embeds `JetBrainsMonoNLNerdFontMono-Regular.ttf` and
`JetBrainsMonoNLNerdFontMono-Bold.ttf` from Nerd Fonts v3.4.0, derived from
JetBrains Mono.
